# Руководство по использованию критерия Стьюдента в GUI

## Описание

Критерий Стьюдента (Student's t-test) используется для сравнения средних значений двух независимых выборок и определения, являются ли они статистически различными.

## Типы критерия Стьюдента

### 1. Равные дисперсии (Classical t-test)
Используется когда дисперсии двух выборок равны. Это классический вариант критерия Стьюдента.

### 2. Неравные дисперсии (Welch's t-test)
Используется когда дисперсии двух выборок различаются. Также известен как критерий Уэлча.

### 3. Автоматический выбор
Программа автоматически выполняет F-критерий Фишера для проверки равенства дисперсий, а затем применяет соответствующий вариант t-критерия.

## Как использовать в GUI

### Шаг 1: Подготовка данных
Создайте два файла с данными для двух выборок:
- `input/data_sample1.txt` - первая выборка
- `input/data_sample2.txt` - вторая выборка

Каждый файл должен содержать числовые значения, по одному на строку.

### Шаг 2: Запуск анализа
1. Откройте GUI приложение: `./gui/build/StatisticalAnalysisGUI.app`
2. Выберите тип анализа "Student Test" из выпадающего меню
3. Выберите метод:
   - "Equal Variance" для равных дисперсий
   - "Unequal Variance" для неравных дисперсий (Уэлч)
   - "Auto" для автоматического выбора
4. Установите уровень значимости α (по умолчанию 0.05)
5. Загрузите первую выборку
6. Загрузите вторую выборку
7. Нажмите "Run Analysis"

### Шаг 3: Просмотр результатов

#### Текстовые результаты
В окне результатов отобразится:
- Размеры выборок (n₁, n₂)
- Средние значения (x̄₁, x̄₂)
- Стандартные отклонения (s₁, s₂)
- t-статистика
- Степени свободы
- Критическое значение
- P-значение
- Результат проверки гипотезы H₀: μ₁ = μ₂

#### Файлы с результатами
Результаты сохраняются в `output/`:
- `student_test_equal_var.txt` - результаты для равных дисперсий
- `student_test_unequal_var.txt` - результаты для неравных дисперсий
- `student_test_auto.txt` - результаты автоматического выбора

#### Графики
Создаются графики распределения Стьюдента с:
- Кривой распределения
- Критическими областями (красным цветом)
- Критическими значениями (пунктирные линии)
- Наблюдаемой t-статистикой (зеленая - H₀ не отвергается, красная - H₀ отвергается)

Файлы графиков:
- `output/plot_student_equal_var.png`
- `output/plot_student_unequal_var.png`
- `output/plot_student_auto.png`

## Интерпретация результатов

### H₀ не отвергается (средние равны)
- |t| ≤ t_critical
- p-value ≥ α
- **Вывод**: Нет статистически значимых различий между средними значениями двух выборок

### H₀ отвергается (средние различаются)
- |t| > t_critical
- p-value < α
- **Вывод**: Средние значения двух выборок статистически различаются

## Пример использования

### Данные
**Выборка 1** (data_sample1.txt):
```
100.7, 98.3, 103.2, 99.5, 102.1, 101.4, 97.8, 104.5, 100.2, 99.1
```

**Выборка 2** (data_sample2.txt):
```
101.5, 104.3, 97.6, 105.2, 99.8, 102.7, 100.1, 98.9, 103.6, 101.3
```

### Ожидаемый результат
При α = 0.05:
- Средние выборок примерно одинаковы
- H₀ не отвергается
- p-value > 0.05

## Рекомендации

1. **Размер выборки**: Для надежных результатов рекомендуется минимум 5-10 наблюдений в каждой выборке
2. **Нормальность**: Критерий Стьюдента предполагает, что данные распределены нормально
3. **Независимость**: Выборки должны быть независимыми друг от друга
4. **Выбор метода**: При неуверенности используйте автоматический выбор (Auto)

## Техническая информация

### Формулы

**Классический t-критерий (равные дисперсии)**:
```
t = (x̄₁ - x̄₂) / (sp * √(1/n₁ + 1/n₂))
где sp² = ((n₁-1)s₁² + (n₂-1)s₂²) / (n₁+n₂-2)
df = n₁ + n₂ - 2
```

**Критерий Уэлча (неравные дисперсии)**:
```
t = (x̄₁ - x̄₂) / √(s₁²/n₁ + s₂²/n₂)
df = (s₁²/n₁ + s₂²/n₂)² / ((s₁²/n₁)²/(n₁-1) + (s₂²/n₂)²/(n₂-1))
```

### Используемые библиотеки
- Boost.Math для вычисления квантилей распределения Стьюдента
- Qt для GUI
- Python/Matplotlib для визуализации

## Решение проблем

### Ошибка "требуется две выборки"
**Решение**: Убедитесь, что загружены оба файла с данными

### График не отображается
**Решение**:
1. Проверьте, что установлен Python и виртуальное окружение
2. Проверьте путь к `python/venv/bin/python3`
3. Убедитесь, что установлены необходимые Python библиотеки (matplotlib, numpy, scipy)

### Неверные результаты
**Решение**:
1. Проверьте формат входных данных (числа, по одному на строку)
2. Убедитесь, что данные не содержат пропущенных значений
3. Проверьте, что выборки независимы и нормально распределены
